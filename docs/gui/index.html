How the Studio Works

                
                

                    
                    

                        
                        

                            
Tokenize

                            

                                The tokenizer breaks down your search query into fundamental components (tokens), such as fields (TI, AB), operators (AND, OR), and search terms.
                            

                        

                    


                    

                        
                        

                            
Lint & Normalize

                            

                                The linter checks the query structure against the rules of the selected source platform, providing suggestions and a 'normalized' version of the query.
                            

                        

                    


                    

                        
                        

                            
Translate

                            

                                Select a target platform and translate your query instantly, handling platform-specific differences in field codes, operators, and syntax.
